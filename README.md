# Мой дневник

**Автор:** Быргазва Милена
**Дата сдачи:** 25.04.2026

## Описание
Программа для ведения личного дневника. Позволяет добавлять заметки с датой, заголовком и текстом, фильтровать их по заголовку.

## Требования
- Python 3.10+
- Tkinter (встроен в Python)

## Как запустить
```bash
python main.py
